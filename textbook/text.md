## 用語整理（重要）

| 用語 | 意味 |
|----|----|
| クラス | 設計図（生徒の型） |
| インスタンス | 設計図から作った実物 |
| 属性 | オブジェクトが持つ情報 |
| メソッド | オブジェクトができる動作 |

---

## クラスの例：Student

```python
class Student:
    def __init__(self, name, grade):
        self.name = name
        self.grade = grade
        self.attendance = False

    def attend(self):
        self.attendance = True

    def absent(self):
        self.attendance = False
