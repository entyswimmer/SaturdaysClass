## 1. リストの作成方法

```python
list1 = [0, 1, 2, 3, 4, 5]
list2 = [50, 20, 30, 40, 10]
list3 = [[0, 1], [1, 2],[3, 4]]
num = '0123456789'
```

リストの作成は上のように行われます。list3は多次元リストと言い、複数のリストを1つのリストとしてまとめます。
また、numのように、`''`で囲まれる文字列はリストとして扱うことが可能です。
リストの各要素の番号は0番目から数えることに注意しましょう。

以下はリストに対しての操作のチートシート(早見表)です。

| 操作内容 | コード例 | 説明 |
|-----------|-----------|------|
| 空のリストの作成 | `[]` | 例a = []|
| リストの長さ | `len(list1)` | 要素数を取得 |
| 最大値 | `max(list1)` | リスト内の最大値を取得 |
| 最小値 | `min(list1)` | リスト内の最小値を取得 |
| 合計値 | `sum(list1)` | リスト内の数値の合計を取得 |
| リストの繰り返し | `list1 * 3` | `list1`を3回繰り返した新しいリストを作成 |
| 要素の存在確認 | `0 in list2` | `0`が`list2`内に存在すればTrueを返す |
| 昇順ソート | `list2.sort()` | リストを昇順に並べ替える（破壊的操作） |
| 降順ソート | `list2.sort(reverse=True)` | リストを降順に並べ替える（破壊的操作） |
| スライス | `num[0:3]` | numの0番目から2番目まで取り出す(012) |
| 要素の追加 | `list2.append(x)` | リストの最後に要素`x`を追加 |
| リストの結合 | `list2.extend([a, b, c])` | `list2`の末尾に複数要素を追加 |
| 指定位置に挿入 | `list2.insert(index, x)` | 指定した`index`位置に要素`x`を挿入 |
| 指定要素の削除 | `list2.remove(x)` | 最初に見つかった要素`x`を削除 |
| インデックス指定削除 | `list2.pop(index)` | 指定した`index`の要素を削除し、戻り値として返す |
| リストのコピー | `list2.copy()` | `list2`の浅いコピーを作成（元の変更は反映されない） |
| 多次元リストの抽出 | `[i[0] for i in list3[:2]]` | `list3`の最初の2つの要素から、各`i`の0番目要素を抽出（例: `[0, 1]`） |

## 2. 関数の作成

```pyhton
#足し算を行う関数
def sum(num1, num2):
    return num1 + num2

sum(1, 2) # 3が出力される

#別の出力方法
ans = sum(1, 2)
print(f"1+2の結果は{ans}です")

#出力
"1+2の結果は3です"
```

- 基本の使い方
    関数は`def 関数名(引数1, 引数2, ・・・):`で宣言します。使用方法は`関数名(引数1, 引数2, ・・・)`です。

- 変数について
    関数宣言の際に使用した引数名はローカル変数といい、関数内部のみで使用可能です。(スコープとは, 変数が使用可能な場所の範囲のことです)
    Pythonでは変数名の型宣言が不要なため、これを気にすることはあまりありません。
    しかし、他の言語、例えばwebサイトに動きをつける言語であるJavaScriptではローカル変数を定義する際は`let num1 = 1`のように宣言する必要があります。今後他の言語に触れる時に思い出してください。(実際にアプリを作成するときはJavaScript(web, モバイルアプリ), Swift(apple製品用アプリ)等がよく用いられます。(変数の型宣言が必要))

- 戻り値について
    関数の戻り値についてですが, `return`という宣言が必要です。何も出さない場合は何も書かないか、`return None`と書きます。
    print()関数で結果の出力は可能ですが, コード内で結果を使うことはできません。
